## Features

* SPA(Single Page Application)
* Image Gallery made automatically with JavaScript
* A lot of ugly CSS to style its look pretty


## Built With

* HTML5
* CSS3 - 향후 Scss로 변경 예정.
* Vanila JavaScript - 되도록이면 ES5를 사용해 IE와 호환하는 것을 목표로 함.
